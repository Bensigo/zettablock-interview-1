## Description
Zettablock Technical test.

## Technology Used
 - Nodejs
 - Nest.js
 - Graphql

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Vist graphql termainl
  Go to http://localhost:3000/graphql to use the graphql api
## Test

```bash
# unit tests
$ npm run test
```
